Android LLD Sample Project

- MVVM
- Multiple Screen
- Unit Testing
- Gson for serialization
- Retrofit for Networking
- Room for Database
- Search feature not there but added commented code MainViewmodel.kt class for reference.
- Added Real network call as well reference code for Fake network call (i.e reading json from assets folder)
- `master` branch - UI in XML, Hilt for DI, without pagination
- `master-compose` branch - UI in Compose, Hilt for DI, without pagination
- `master-pagination` branch - UI in XML, Hilt for DI, with pagination
- `master-compose-koin` branch - UI in Compose, Koin for DI, without pagination
